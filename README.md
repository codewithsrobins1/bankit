## Getting Started

To get a local copy up and running follow these steps using your terminal.

### Installation

1. Clone the repo
```sh
git clone https://github.com/codewithsrobins1/bankit
```
2. Install NPM packages
```sh
npm install i

```
3. Start Client
```sh
npm start

```
